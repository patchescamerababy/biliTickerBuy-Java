#### 兼容性
支持Java8，需要JavaFX，如果启动失败，请使用Java8，理论上兼容WinXP\Win7，Linux/macOS理论上也支持。启动方法

    java -jar biliTickerBuy.jar
    

第一次登录会下载chromium等依赖，速度取决于网络，不懂的直接点登录，然后等待跳出浏览器即可
